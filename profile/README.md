Креафлай - платформа генерации маркетингового контента с помощью AI-агентов. Решение построено как набор микросервисов: генерация контента, управление пользователями и доступом, рассылка, управление секретами и оркестрация MCP серверов.

## Ключевые возможности

- Генерация шаблонов писем и контента на основе задач пользователя
- Генерация контента (аудио / видео / картинки) для опубликования в соц. сетях
- Генерация текста для формирования постов в соц. сетях
- Отправка писем через SMTP
- Безопасное хранение секретов через HashiCorp Vault
- Подключение MCP серверов для расширения возможностей AI-агентов
- Оркестрация MCP серверов для AI-инструментов на тенанта

## Стек

**Frontend**: Next.js (TypeScript), Tailwind CSS, Shadcn, TanStack React Query, Framer Motion, Zod
**Backend**:

- AI-сервис на Fastify (TypeScript), React Email, LangChain
- Основные сервисы на Golang (gin + pgx)
- MCP-оркестратор на Java (Quarkus, jOOQ)
- MCP-сервера на Python (fastmcp)

## Open Source

Часть проекта является Open Source, на данный момент это включает в себя все доступные MCP сервера с платформы.
